1. There are many tabs: Elements, Console, Source, Network etc. You have to click on Network tab.
  2. Record button. Red is on. Gray is off. It has to be on (Red).
  3. Clear button. It clears the request logs. Click on it before visiting a player.
  4. Look for a request with type json. Click on it.
  5. Request details are shown on the right hand side. Again there are many tabs. Click on Response tab.
  6. Verify you are looking at correct response. Search "OtherPlayerService". If it is there you can search for other thing. If not start again from step 3.
